# Calculator
Calculator WPF Application
  
Simple Calculator Application made with WPF  
  
Keyboard shortcuts:  
All number buttons are binded to their respective positions on the keyboard. Same goes for Shift+8 for multiply, etc.  
Other shortcuts:  
s = 'sqrt'  
r = 'root'  
m = 'mod'  
n = '+/-'  
Backspace = 'BCK'  
Enter = '='
